# Hashr

Hashr is a simple Python script that allows users to compare the checksum of a file with a given checksum. It supports multiple hash types like MD5, SHA1, SHA256, and SHA512.

## Features

- Select a file to check its checksum.
- Enter a published checksum to compare.
- Supports common hash types: `MD5`, `SHA1`, `SHA256`, `SHA512`.
- Easy-to-use graphical interface (Tkinter).
- Cross-platform support: Works on Linux, macOS, and Windows.

## Requirements

- Python 3.x
- tkinter (usually comes with Python)
- hashlib (part of the Python Standard Library)

## Supported Platforms

Hashr is designed to work across multiple platforms:

- Linux
- macOS
- Windows
